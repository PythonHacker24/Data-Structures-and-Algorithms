Maximum Depth of Binary Tree

Time: O(n) Space: O(h)

üí° Approach

Use DFS (recursion) to find the maximum depth. The depth of a tree is 1 + max depth of its subtrees.

üìù Steps

  1. Base case: if root is None, return 0
  2. Recursively find depth of left subtree
  3. Recursively find depth of right subtree
  4. Return 1 + maximum of left and right depths
def maxDepth(self, root):
    if not root:
        return 0
    
    ml = self.maxDepth(root.left)
    mr = self.maxDepth(root.right)
    
    return 1 + max(ml, mr)

Binary Tree Inorder Traversal

Time: O(n) Space: O(h)

üí° Approach

Inorder: Left ‚Üí Root ‚Üí Right. Use recursion to traverse and collect values.

üìù Steps

  1. Traverse left subtree
  2. Visit root (add to result)
  3. Traverse right subtree
def inorderTraversal(self, root):
    if not root:
        return []
    
    self.inorderTraversal(root.left)
    self.results.append(root.val)
    self.inorderTraversal(root.right)
    
    return self.results
üí° Pro Tip: For iterative approach, use a stack. Push all left nodes first, then process and go right.

Binary Tree Level Order Traversal

Time: O(n) Space: O(w)

üí° Approach

Use BFS with a queue. Process each level completely before moving to the next.

üìù Steps

  1. Initialize queue with root
  2. For each level, process all nodes currently in queue
  3. Add children to queue for next level
  4. Store each level as a separate list
def levelOrder(self, root):
    queue = deque()
    result = []
    
    if root:
        queue.append(root)
    
    while len(queue) > 0:
        level = []
        for i in range(len(queue)):
            curr = queue.popleft()
            level.append(curr.val)
            if curr.left:
                queue.append(curr.left)
            if curr.right:
                queue.append(curr.right)
        result.append(level)
    
    return result

Balanced Binary Tree

Time: O(n) Space: O(h)

üí° Approach

Check balance condition while calculating depth. A tree is balanced if height difference between subtrees ‚â§ 1.

def maxDepth(self, root):
    if not root:
        return 0, True
    
    ld, leftBal = self.maxDepth(root.left)
    rd, rightBal = self.maxDepth(root.right)
    
    if not leftBal or not rightBal or abs(ld - rd) > 1:
        return 1 + max(ld, rd), False
    
    return 1 + max(ld, rd), True

Validate Binary Search Tree

Time: O(n) Space: O(h)

üí° Approach

Use DFS with bounds. Each node must be within valid range based on ancestors.

üìù Steps

  1. Start with infinite bounds
  2. For left child: update max bound to current node value
  3. For right child: update min bound to current node value
  4. Check if current node violates bounds
def validate(self, root, minVal, maxVal):
    if not root:
        return True
    
    if (root.val <= minVal) or (root.val >= maxVal):
        return False
    
    return (self.validate(root.left, minVal, root.val) and 
            self.validate(root.right, root.val, maxVal))

Same Tree

Time: O(n) Space: O(h)

üí° Approach

Recursively compare nodes. Check structure and values simultaneously.

def isSameTree(self, p, q):
    if not p and not q:
        return True
    
    if (not p and q) or (not q and p):
        return False
    
    return (p.val == q.val and 
            self.isSameTree(p.left, q.left) and
            self.isSameTree(p.right, q.right))

Invert Binary Tree

Time: O(n) Space: O(h)

üí° Approach

Recursively swap left and right children of each node.

def invertTree(self, root):
    if not root:
        return
    
    temp = root.left
    root.left = root.right
    root.right = temp
    
    self.invertTree(root.left)
    self.invertTree(root.right)
    
    return root

Construct Binary Tree from Preorder and Inorder

Time: O(n) Space: O(n)

üí° Approach

Use first element of preorder as root, find it in inorder to split left/right subtrees.

üìù Steps

  1. First element in preorder is always root
  2. Find root in inorder to determine left/right subtrees
  3. Split preorder and inorder accordingly
  4. Recursively build left and right subtrees
def buildTree(self, preorder, inorder):
    if not preorder or not inorder:
        return None
    
    root = TreeNode(preorder[0])
    root_index = inorder.index(preorder[0])
    
    root.left = self.buildTree(preorder[1:root_index + 1], 
                                   inorder[:root_index])
    root.right = self.buildTree(preorder[root_index + 1:], 
                                    inorder[root_index + 1:])
    
    return root

Search in BST

Time: O(h) Space: O(h)

üí° Approach

Use BST property: left < root < right. Navigate based on comparison.

def searchBST(self, root, val):
    if not root:
        return None
    
    if val > root.val:
        return self.searchBST(root.right, val)
    elif val < root.val:
        return self.searchBST(root.left, val)
    else:
        return root

Insert into BST

Time: O(h) Space: O(h) 
def insertIntoBST(self, root, val):
    if not root:
        return TreeNode(val)
    
    if val > root.val:
        root.right = self.insertIntoBST(root.right, val)
    if val < root.val:
        root.left = self.insertIntoBST(root.left, val)
    
    return root

Delete Node in BST

Time: O(h) Space: O(h)

üí° Approach

Three cases: 1) No children - delete directly, 2) One child - replace with child, 3) Two children - replace with inorder successor.

def deleteNode(self, root, key):
    if not root:
        return None
    
    if key > root.val:
        root.right = self.deleteNode(root.right, key)
    elif key < root.val:
        root.left = self.deleteNode(root.left, key)
    else:
        if not root.right:
            return root.left
        elif not root.left:
            return root.right
        else:
            mNode = self.minNode(root.right)
            root.val = mNode.val
            root.right = self.deleteNode(root.right, mNode.val)
    
    return root

def minNode(self, root):
    curr = root
    while curr and curr.left:
        curr = curr.left
    return curr

Diameter of Binary Tree

Time: O(n) Space: O(h)

üí° Approach

Diameter = longest path between any two nodes. For each node, diameter = left_height + right_height.

def diameterOfBinaryTree(self, root):
    self.diameter = 0
    
    def maxDepth(root):
        if not root:
            return 0
        
        ld = maxDepth(root.left)
        rd = maxDepth(root.right)
        
        self.diameter = max(self.diameter, ld + rd)
        
        return 1 + max(ld, rd)
    
    maxDepth(root)
    return self.diameter

Binary Tree Right Side View

Time: O(n) Space: O(w)

üí° Approach

Use level-order traversal. For each level, take the last node processed.

def rightSideView(self, root):
    queue = deque()
    if root:
        queue.append(root)
    
    result = []
    while len(queue) > 0:
        for i in range(len(queue)):
            curr = queue.popleft()
            if curr.left:
                queue.append(curr.left)
            if curr.right:
                queue.append(curr.right)
            rightmost = curr.val  # Last node in level
        result.append(rightmost)
    
    return result

Count Good Nodes in Binary Tree

Time: O(n) Space: O(h)

üí° Approach

A node is "good" if its value ‚â• maximum value on path from root. Use DFS with running max.

def goodNodes(self, root):
    def preorder(root, maxVal):
        if not root:
            return 0
        
        count = 1 if root.val >= maxVal else 0
        newMax = max(maxVal, root.val)
        
        count += preorder(root.left, newMax)
        count += preorder(root.right, newMax)
        
        return count
    
    return preorder(root, float('-inf'))

Subtree of Another Tree

Time: O(m√ón) Space: O(h)

üí° Approach

For each node in main tree, check if subtree starting there matches target. Reuse "Same Tree" logic.

def isSubtree(self, root, subRoot):
    if not root:
        return False
    
    if self.isSameTree(root, subRoot):
        return True
    
    return (self.isSubtree(root.left, subRoot) or 
            self.isSubtree(root.right, subRoot))

Lowest Common Ancestor of BST

Time: O(h) Space: O(h)

üí° Approach

Use BST property: if both nodes are in different subtrees of current node, current node is LCA.

def lowestCommonAncestor(self, root, p, q):
    if not root or root == p or root == q:
        return root
    
    left = self.lowestCommonAncestor(root.left, p, q)
    right = self.lowestCommonAncestor(root.right, p, q)
    
    if not right:
        return left
    elif not left:
        return right
    else:
        return root

üß† Key Patterns & Mental Models

üéØ Common Patterns

  • DFS Recursive: Most tree problems can be solved with recursive DFS
  • Level Order (BFS): Use queue for level-by-level processing
  • Two-Pass Strategy: Sometimes solve subproblems first, then combine
  • Global Variables: Use class variables for values that need to persist across recursive calls
  • BST Properties: Left < Root < Right - exploit this for O(h) operations

üîÑ Tree Traversal Methods

üìù Traversal Types

  1. Preorder: Root ‚Üí Left ‚Üí Right (useful for copying trees)
  2. Inorder: Left ‚Üí Root ‚Üí Right (gives sorted order in BST)
  3. Postorder: Left ‚Üí Right ‚Üí Root (useful for deleting trees)
  4. Level Order: Level by level using queue (BFS)

üí° When to Use Each

DFS: When you need to go deep first (height, validation, path problems)
BFS: When you need level information (level order, right side view, minimum depth)

‚ö° Time & Space Complexity Guide

DFS: O(n) time, O(h) space BFS: O(n) time, O(w) space

Where: n = number of nodes, h = height of tree, w = maximum width of tree

üí° Space Complexity Notes:
‚Ä¢ Balanced tree: h = O(log n)
‚Ä¢ Skewed tree: h = O(n)
‚Ä¢ BFS space depends on widest level

üõ†Ô∏è Common Techniques

üìù Problem-Solving Strategies

  1. Base Case First: Always handle null nodes
  2. Divide & Conquer: Solve for subtrees, then combine
  3. Pass Information Down: Use parameters to pass context (bounds, path info)
  4. Collect Information Up: Return values from recursive calls
  5. Use Helper Functions: Keep main function clean

üö® Common Mistakes to Avoid

‚ö†Ô∏è Watch Out For

  • Forgetting to handle null/empty tree cases
  • Not returning the modified tree in modification problems
  • Using class variables without resetting them
  • Confusing tree height vs depth (usually same, but context matters)
  • In BST problems, forgetting that duplicates matter
  • Stack overflow on deeply nested trees

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# Basic DFS Template
def dfs(root):
    if not root:
        return  # base case
    
    # Process current node
    dfs(root.left)   # recurse left
    dfs(root.right)  # recurse right

# Basic BFS Template
def bfs(root):
    if not root:
        return
    
    queue = deque([root])
    while queue:
        node = queue.popleft()
        # Process node
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
